# urban-canopy-detection

> Tree-crown detection on aerial imagery: modern detectors (YOLO26, RF-DETR) evaluated against a published RetinaNet baseline on the open NEON benchmark — leakage-safe geographic splits, SAHI tiled inference, GeoJSON/GeoPackage outputs, and a live demo.

🚧 **Modernization in progress (July 2026).** Originally built Fall 2024 as an independent YOLOv5 project; being published and upgraded here with honest metrics, tests, and CI. Code landing soon.
